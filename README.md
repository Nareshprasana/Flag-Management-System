# Feature Flag Management System

## Applications

1. Backend API
2. Super Admin App
3. Organization Admin App
4. End User App

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Next.js
- Tailwind CSS
- JWT Authentication

## Run Backend

cd backend
npm install
npm run dev

## Run Super Admin App

cd super-admin-app
npm install
npm run dev

## Run Admin App

cd admin-app
npm install
npm run dev

## Run End User App

cd user-app
npm install
npm run dev
